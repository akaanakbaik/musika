# Musika

A full-featured Spotify-like music PWA built with React + Vite + Express + Supabase.

## Features
- Multi-source search (YouTube, Spotify, Apple Music, SoundCloud)
- Supabase auth (email + OTP)
- PWA with offline support
- AI Chat (GPT-5)
- Full player with queue, shuffle, repeat
- Playlists, favorites, history
